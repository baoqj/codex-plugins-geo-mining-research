# Bentonite Buffer Reaction Network

Core processes:

- Cation exchange.
- Surface complexation / sorption.
- Diffusion-dominated transport.
- Thermal gradient effects.
- Porewater chemistry evolution.
- Mineral dissolution/precipitation.
- Radionuclide diffusion and sorption.
- Swelling and hydraulic conductivity evolution if M is active.

Required data: bentonite composition, exchange capacity, swelling pressure, porewater chemistry, temperature profile, radionuclide inventory, diffusion coefficients, mineral transformation constraints.
