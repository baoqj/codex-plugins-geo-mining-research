# Geothermal Water-Rock Reaction Network

Core processes:

- Temperature-dependent equilibrium.
- Silica scaling.
- Carbonate precipitation.
- Metal mobility.
- Fluid-rock dissolution.
- Injection/production mixing.
- Fracture permeability change from scaling or dissolution.

Required data: temperature, pressure, flow rate, fluid chemistry, mineralogy, injection chemistry, production chemistry, scaling observations, fracture permeability.
