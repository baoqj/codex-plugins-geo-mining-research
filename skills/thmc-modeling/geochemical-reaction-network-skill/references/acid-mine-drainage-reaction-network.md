# Acid Mine Drainage Reaction Network

Core processes:

- Pyrite and sulfide oxidation.
- Fe2+/Fe3+ redox cycling.
- Sulfate and acidity generation.
- Carbonate neutralization.
- Fe/Al hydroxide precipitation.
- Heavy metal mobilization.
- Secondary sulfate minerals.

Required data: sulfide mineralogy, oxygen availability, flow path, pH, Eh, sulfate, Fe, Al, trace metals, alkalinity, temperature, particle size or surface area if kinetics are modeled.
