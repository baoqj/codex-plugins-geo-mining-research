# Tailings Seepage Reaction Network

Core processes:

- Leachate generation.
- Sulfate and metal transport.
- Evaporation concentration.
- Acid generation or neutralization.
- Secondary mineral formation.
- Sorption/desorption in tailings and receiving aquifer.
- Consolidation or permeability evolution if HMC/THMC is active.

Required data: tailings mineralogy, porewater chemistry, seepage rates, seasonal temperature, water table, hydraulic conductivity, pH, Eh, sulfate, metals, alkalinity.
