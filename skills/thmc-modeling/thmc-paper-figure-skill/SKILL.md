# THMC Paper Figure Skill

## Purpose

Create a paper-ready figure plan for a THMC Modeling Package. Use `academic-figure-package-skill` when a complete Figure Package is requested.

## Default Figure Set

- Figure 1: conceptual THMC model.
- Figure 2: coupling matrix and feedback pathways.
- Figure 3: model geometry and boundary conditions.
- Figure 4: geochemical reaction network.
- Figure 5: calibration and validation plots.
- Figure 6: spatial-temporal simulation outputs.
- Figure 7: sensitivity and uncertainty results.
- Optional graphical abstract.

## Output Contract

Return:

- Figure package outline.
- Panel-by-panel plan.
- Visual grammar.
- Data and source requirements.
- Caption drafts.
- Publication checklist.
- Caveats for conceptual, uncalibrated, or simulated figures.
