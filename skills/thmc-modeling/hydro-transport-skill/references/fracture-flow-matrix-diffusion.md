# Fracture Flow and Matrix Diffusion

Use fracture-flow concepts when transport is controlled by connected fractures, preferential pathways, or low-permeability rock matrix storage.

Options:

- Equivalent porous medium.
- Dual-porosity / dual-permeability.
- Discrete fracture network.
- Mobile-immobile mass transfer.

Important data:

- Fracture aperture.
- Fracture spacing.
- Connectivity.
- Matrix porosity and diffusion.
- Sorption in matrix.
- Tracer or breakthrough observations.
