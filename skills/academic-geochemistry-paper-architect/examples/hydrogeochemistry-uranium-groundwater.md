# Example: Uranium Groundwater Hydrogeochemistry Paper Architecture

## Input Topic

Study uranium migration in fractured Canadian Shield groundwater using water chemistry and PHREEQC modelling.

## Paper Type Classification

- Primary type: Hydrogeochemistry Paper
- Secondary type: Reactive Transport Modelling Paper
- Tertiary type: Radiolysis / Nuclear Geochemistry Paper

## Research Questions

- Which water-rock, redox, carbonate, and mineral-saturation controls govern uranium mobility?
- Are observed uranium concentrations consistent with U(VI)-carbonate complexation, mineral equilibrium, or mixing?
- Which interpretations remain uncertain without site-specific flow and mineralogical constraints?

## Hypotheses

- If carbonate complexation controls uranium mobility, uranium should covary with alkalinity, pH/Eh state, and PHREEQC U(VI)-carbonate speciation.
- If mineral equilibrium controls uranium, saturation indices for relevant U-bearing phases should constrain solubility.

## Required Data

- Water sample metadata, pH, Eh, temperature, alkalinity, major ions, U, Fe, sulfate, carbonate species if available.
- Charge-balance inputs, detection limits, field/lab method notes, PHREEQC database selection, mineralogical context.

## Methods

- Charge balance and data audit.
- Piper or equivalent water-type classification.
- PHREEQC speciation and saturation-index analysis.
- Alternative-explanation review: lithology background, mixing, sampling bias, redox disequilibrium.

## Figure Plan

- Study area and hydrogeological setting map.
- Piper diagram.
- Eh-pH diagram.
- Uranium speciation fraction plot.
- Saturation-index plot.
- Conceptual model of uranium mobility.
