# Figure and Table Architecture

Every figure must support a specific paper claim. Do not add decorative figures.

## Required Figure Metadata

For each figure specify:

- figure id and paper section;
- claim supported;
- data source, version, and processing step;
- variables, axes, units, uncertainty, and detection-limit handling;
- measured/modelled/inferred/illustrative status;
- caption message and caveat.

## General Figures

- Study area map
- Geological map
- Sampling map
- Data workflow diagram
- QA/QC flowchart
- Histogram
- Boxplot
- Correlation matrix
- PCA biplot
- Spatial heatmap
- Conceptual model

## Rock Geochemistry Figures

- TAS diagram
- Harker diagrams
- REE patterns
- Spider diagrams
- Tectonic discrimination diagrams
- Isotope evolution diagrams
- Age histogram or density plot

## Hydrogeochemistry Figures

- Piper diagram
- Stiff diagram
- Gibbs diagram
- Eh-pH diagram
- Saturation-index plot
- Speciation plot
- Reaction path diagram
- Breakthrough curve

## Exploration Geochemistry Figures

- Anomaly map
- Target ranking map
- Multi-layer GIS overlay
- Feature-importance plot
- ROC/AUC only when valid labels exist
- Prospectivity score table

## Radiolysis and Nuclear Geochemistry Figures

- Radiolysis reaction network
- G-value comparison
- Pore-size effect plot
- Dose-rate effect plot
- Electric-field effect plot
- Surface band alignment
- Bulk water versus porous water comparison table

## Table Plan

Common tables include:

- sample metadata and analytical methods;
- data source and version table;
- QA/QC and detection-limit table;
- derived-index table;
- model parameter table;
- uncertainty and limitation table;
- target ranking table for exploration papers.
