# Flow And Transport Mode Selection

Use PFLOTRAN when the question requires spatial flow and transport rather than single-water chemistry.

## Mode Rules

- Saturated groundwater: use when the domain remains fully saturated and hydraulic head/pressure boundaries are defensible.
- Variably saturated flow: use for tailings covers, vadose zones, infiltration, seepage faces, or changing saturation.
- Advection-dispersion transport: use for plume movement, breakthrough curves, and field-scale solute transport.
- Diffusion-dominated transport: use for low-permeability rock, bentonite, matrix diffusion, or repository near-field screening.
- Reactive transport: use when mineral reactions, sorption, ion exchange, or chemistry feedback is central.

## Do Not Invent

Hydraulic gradients, recharge, seepage rates, permeability fields, dispersivity, diffusion coefficients, or source fluxes.
