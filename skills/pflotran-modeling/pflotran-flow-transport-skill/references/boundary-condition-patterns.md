# Boundary Condition Patterns

| Boundary type | Use case | Required data |
| --- | --- | --- |
| Dirichlet pressure/head | fixed upstream/downstream head | measured or scenario head/pressure |
| Neumann flux | recharge, seepage, injection, extraction | measured flux or clearly stated scenario |
| No-flow | symmetry or impermeable boundary | geologic justification |
| Concentration/source | contaminant or porewater input | source chemistry and flux/timing |
| Heat boundary | geothermal, waste heat, mine heat | temperature or heat flux |

Every boundary condition must state source, uncertainty, and whether it is measured, assumed, or placeholder.
