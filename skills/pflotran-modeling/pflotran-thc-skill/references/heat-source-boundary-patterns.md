# Heat Source And Boundary Patterns

| Pattern | Required data |
| --- | --- |
| Geothermal gradient | gradient, basal/side temperature, thermal properties |
| Waste heat source | heat rate, geometry, time function |
| Mine heat pollution | source temperature, flow rate, duration |
| Surface seasonal boundary | time series, amplitude, phase |

Heat boundaries must be tagged measured, literature-based, scenario, or placeholder.
