# Acid Mine Drainage Pattern

Key processes:

- sulfide oxidation source terms;
- acidity, sulfate, Fe, Al, Mn, and metal release;
- carbonate neutralization;
- Fe/Al hydroxide and sulfate mineral precipitation;
- sorption on secondary Fe/Al phases;
- porosity/permeability feedback if precipitation is material.

Do not invent pyrite content, oxygen flux, sulfide oxidation rates, or neutralization capacity.
