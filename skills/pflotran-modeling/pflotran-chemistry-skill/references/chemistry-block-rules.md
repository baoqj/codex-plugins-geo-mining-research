# Chemistry Block Rules

PFLOTRAN chemistry plans should list primary species, secondary species, minerals, kinetic minerals, sorption/exchange reactions, gases if relevant, and database requirements.

Start from a PHREEQC prototype when available, but do not assume PHREEQC syntax maps one-to-one to PFLOTRAN. Preserve reaction intent, species list, mineral hypotheses, and required constants.

All undocumented constants must remain placeholders.
