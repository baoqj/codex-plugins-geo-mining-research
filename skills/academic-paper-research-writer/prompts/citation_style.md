Apply strict academic citation discipline.

Supported citation styles:
- ACS;
- APA;
- Nature;
- Elsevier numbered style;
- IEEE.

Rules:
1. Every non-trivial factual claim requires a citation or an explicit `Citation needed:` note.
2. Do not invent references.
3. If source metadata is incomplete, mark it as incomplete rather than fabricating details.
4. Prefer primary literature, peer-reviewed reviews, government datasets, and official technical reports.
5. For rapidly changing data, mark that current verification is required.
6. For equations, cite the source if standard, or label as derived if derived in the paper.
7. Use one citation style consistently.
8. Include DOI where available.
9. Distinguish between cited evidence and the paper's own inference.
