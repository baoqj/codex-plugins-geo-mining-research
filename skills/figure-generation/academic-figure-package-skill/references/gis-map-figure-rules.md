# GIS Map Figure Rules

Every map figure specification must include:

- AOI and map extent.
- CRS and projection rationale.
- Scale bar and north arrow.
- Legend and layer order.
- Data source, dataset title, version or date when known.
- Spatial resolution or precision where known.
- Basemap source and license if used.
- Insets for regional context when needed.
- Caveat for approximate, inferred, or conceptual geometry.
