# Publication Checklist

- Figure intent is clear and tied to a paper section.
- Main message is stated in one sentence.
- Panel labels are present and referenced in the caption.
- Fonts, line weights, symbols, and colors are consistent.
- Legend explains all visual encodings.
- Data sources and methods are cited.
- Data plots are reproducible from scripts or documented steps.
- GIS maps include CRS, scale bar, north arrow, legend, source, and spatial precision.
- Geochemistry figures include medium, units, analytical method, detection limits, thresholds, and QA/QC limits.
- Mining or NI 43-101 figures include QP/legal/investment/data caveats where relevant.
- Output format matches intended use.
- Raster images meet expected resolution at print size.
