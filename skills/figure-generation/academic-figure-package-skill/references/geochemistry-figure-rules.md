# Geochemistry Figure Rules

Geochemistry figures must specify:

- Sample medium.
- Elements and units.
- Analytical method and laboratory if known.
- Detection limits and censored-data handling.
- QA/QC status and limitations.
- Threshold method for anomalies.
- Compatibility caveat when comparing surveys.
- Deposit-model pathfinder interpretation boundary.

Do not infer mineralization or economic value from geochemical anomalies alone.
