# Vector vs Raster

Prefer vector formats for diagrams, maps with simple geometry, line art, labels, and publication layouts.

Use raster formats for imagery, hillshade, dense heatmaps, rendered 3D scenes, or photographic content.

Recommended exports:

- Diagrams: SVG and PDF.
- Maps: PDF/SVG for vector layers, TIFF/PNG for raster-heavy maps.
- Data plots: PDF/SVG plus PNG preview.
- Web/3D renders: PNG for snapshot, HTML for interactive artifact.

For raster submission, target at least 300 dpi at final print size unless the journal specifies otherwise.
