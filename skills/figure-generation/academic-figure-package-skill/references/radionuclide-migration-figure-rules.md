# Radionuclide Migration Figure Rules

- Show radionuclide series and decay relationships clearly.
- Separate decay, sorption, advection, dispersion, diffusion, precipitation, and gas transport processes.
- Mark conceptual pathways as conceptual.
- Include hydrogeologic units, flow direction, redox zones, and sampling points when relevant.
- Include safety, regulatory, and measurement caveats.
- Do not present a conceptual migration figure as a site-specific risk assessment without verified data.
