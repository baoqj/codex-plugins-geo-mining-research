# Data Visualization Rules

- Define the plotted variable, unit, transformation, filter, and missing-data treatment.
- State whether data are measured, modeled, inferred, or illustrative.
- Use appropriate axes, legends, and uncertainty bars.
- Do not invent sample counts, assay values, model scores, or statistical significance.
- Use robust thresholds only when the population and method are documented.
- Preserve script path and parameters for reproducibility.
