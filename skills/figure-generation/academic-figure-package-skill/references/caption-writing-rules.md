# Caption Writing Rules

Every caption should include:

- What the figure shows.
- What each panel shows.
- Data source and provenance.
- Method or calculation summary.
- Symbol, color, and abbreviation meanings.
- Important assumptions, uncertainty, or caveats.

Avoid:

- Overclaiming results not shown in the figure.
- Hiding inferred or conceptual content.
- Treating a conceptual figure as measured geometry.
- Omitting source and method details.
