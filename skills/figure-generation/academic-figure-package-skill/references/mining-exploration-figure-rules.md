# Mining Exploration Figure Rules

- Separate observed evidence from inferred targets.
- Use cautious labels such as prospective corridor, anomaly zone, or follow-up area when evidence supports them.
- Do not label ore, reserve, mineable zone, or economic target without appropriate technical basis.
- Preserve claim status date, tenure source, and uncertainty.
- Show infrastructure distances only when calculated or sourced.
- Add caveats for exploration-stage interpretation.
