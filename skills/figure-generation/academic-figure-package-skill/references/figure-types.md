# Figure Types

- Problem-framing figure: defines the scientific question or knowledge gap.
- Study-area figure: shows AOI, geology, claims, infrastructure, and spatial context.
- Method-workflow figure: shows sampling, analysis, modeling, or AI workflow.
- Experimental-design figure: shows controls, variables, treatments, and measurements.
- Conceptual-mechanism figure: shows processes, pathways, and causal links.
- Data-visualization figure: plots measured or modeled values.
- GIS / map figure: uses spatial layers and cartographic metadata.
- Geochemical-anomaly figure: shows element patterns and thresholds.
- Model-architecture figure: shows computational or agent workflow components.
- Evidence-synthesis figure: summarizes claims, evidence strength, uncertainty, and gaps.
- Technical-report figure: supports disclosure with clear caveats.
- Graphical abstract: summarizes the paper's central contribution.
