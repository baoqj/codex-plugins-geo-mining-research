# Publication Checklist

- [ ] Figure intent supports a specific paper section.
- [ ] Main visual argument is explicit.
- [ ] Panel labels are present and referenced in caption.
- [ ] Font family and size are consistent.
- [ ] Legend explains colors, symbols, arrows, and line styles.
- [ ] Color does not carry meaning alone.
- [ ] Data source and method are cited.
- [ ] Scripts or rendering steps are documented.
- [ ] Map figures include CRS, scale bar, north arrow, legend, source, and spatial precision.
- [ ] Geochemistry figures include sample medium, units, detection limits, threshold method, and QA/QC limitation.
- [ ] Mining figures include QP/legal/investment/data caveats where relevant.
- [ ] Export format and resolution meet target use.
