# Evidence Grading

Use evidence grades to keep mining research outputs auditable.

## Grade A

- Government geological survey datasets with metadata.
- Peer-reviewed papers.
- Signed technical reports.
- QP-supported technical materials.

Use Grade A evidence for primary findings when the source directly supports the claim.

## Grade B

- Assessment reports.
- Historical drilling with usable metadata.
- Company technical presentations with source details.
- Well documented third-party datasets.

Use Grade B evidence for supporting findings, but check date, method, QA/QC, and source purpose.

## Grade C

- Incomplete metadata.
- Unverified secondary summaries.
- Web pages without enough technical context.
- Occurrence descriptions with uncertain coordinates or unclear source lineage.

Use Grade C evidence only as a lead or supporting context.

## Grade D

- Model inference.
- Unverified statistical, remote-sensing, or geophysical anomaly.
- Hypothesis-level target generation.

Use Grade D evidence as a hypothesis only. It cannot independently support a material technical conclusion.

## Conflict Handling

- Prefer the source closest to the original data.
- Prefer newer data only when methods and metadata are stronger.
- Preserve conflicts instead of averaging them away.
- State when conflicts require field verification, QP review, or source-document review.
