# THMC Evidence and Data Requirements

| Data group | Status | Examples |
|---|---|---|
| Geological data | required | lithology, stratigraphy, faults, fractures, domain geometry |
| Hydrogeological data | required | head, hydraulic conductivity, porosity, storage, flow direction |
| Water chemistry | required for C | pH, Eh, EC/TDS, major ions, trace metals, radionuclides |
| Mineralogy | required for reactions | XRD, modal mineralogy, reactive surfaces |
| Thermodynamic database | required for reactions | PHREEQC database or solver database |
| Kinetic parameters | scenario-specific | pyrite oxidation, silicate dissolution, precipitation rates |
| Sorption parameters | scenario-specific | Kd, surface complexation sites, exchange capacity |
| Thermal properties | required for T | thermal conductivity, heat capacity, geothermal gradient |
| Mechanical properties | required for M | Young modulus, Poisson ratio, Biot coefficient, fracture stiffness |
| Field observations | recommended | time series, monitoring wells, tracer tests |
| Laboratory experiments | recommended | batch, column, sorption, kinetic, swelling tests |
| Monitoring data | required for validation | independent wells, chemistry, head, temperature |
